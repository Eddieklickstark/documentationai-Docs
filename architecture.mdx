## Stack

- React + Vite für das Frontend
- Convex für Datenbank, Queries, Mutations, Actions und Cronjobs
- Clerk für Authentifizierung
- HubSpot für CRM-Daten
- Resend für E-Mail Versand
- Vercel für Frontend Deployment

## Wichtige Codebereiche

```text
src/App.tsx                 Haupt-App, Routing und Screens
src/App.css                 UI Styling
src/lib/metrics.ts          Sprint- und Statusmetriken
src/lib/convexDashboard.ts  Frontend-Dashboard-Helfer
convex/dashboard.ts         Kunden-/Admin-Dashboard Queries
convex/access.ts            Invite, Magic Link und Kundensessions
convex/authz.ts             Serverseitige Admin-Rechteprüfung
convex/hubspot.ts           HubSpot Sync und Diagnoseactions
convex/schema.ts            Datenmodell
convex/crons.ts             Stündlicher HubSpot Sync
docs/dashboard.md           Technische Projektdoku im App-Repo
```

## Datenfluss

```text
HubSpot -> Convex HubSpot Actions -> Convex Tabellen -> React Dashboard
Clerk   -> Convex Auth/AuthZ       -> Admin- und Kundenzugriff
Resend  -> Invite/Magic Link Mail  -> Customer Session
```
